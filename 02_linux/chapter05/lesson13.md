<h3>UNIX Philosophy</h3>
<ol>
  <li>
Write programs that do one thing and do it well.
    
  </li>
  <li>
Write programs to work together.
    
  </li>
  <li>
Write programs to handle text streams, because that is a universal interface.
    
  </li>
</ul>
